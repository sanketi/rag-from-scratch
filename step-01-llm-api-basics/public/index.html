Response anatomy — what each field means

      

        

          content[0].text
          
The actual answer. This is the one line you extract in every AI app.

        

        

          usage.input_tokens
          
Tokens counted from your system prompt + all messages. This is what you pay for on input.

        

        

          usage.output_tokens
          
Tokens generated in the reply. Usually costs more per token than input.

        

        

          stop_reason
          
end_turn = natural finish. max_tokens = answer got cut off — raise max_tokens or shorten your prompt.
